# Codex Workspace

This repository is used to track changes made in this workspace so edits can be reviewed, reverted, and restored over time.

## Daily workflow

- Save a snapshot after meaningful changes: `git add . && git commit -m "Describe the change"`
- Check what changed: `git status` and `git diff`
- View history: `git log --oneline`
- Restore a file from the last commit: `git restore path/to/file`
